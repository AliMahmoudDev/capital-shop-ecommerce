# 🛍️ Capital Shop - E-commerce Landing Page

![Project Banner](/assets/images/README-img.png)

> A modern, high-performance fashion e-commerce interface built with **Semantic HTML5**, **CSS3 (BEM methodology)**, and **Vanilla JavaScript**.

---

## 📝 About The Project

**Capital Shop** is a fully responsive landing page designed to showcase best practices in frontend development. The goal was to build a scalable UI structure without relying on heavy frameworks, demonstrating a strong grasp of core web technologies.

This project highlights:
* **Clean Architecture:** Using BEM (Block Element Modifier) for maintainable CSS.
* **Modern Styling:** Utilizing CSS Variables (custom properties) for theming.
* **Performance:** Optimized assets and minimal JavaScript for maximum speed.
* **Smart Workflow:** Leveraging AI tools for initial scaffolding, followed by manual engineering for logic and UX refinement.

---

## ✨ Key Features

* **📱 Fully Responsive:** Seamless experience across Mobile, Tablet, and Desktop.
* **🎨 Modern UI/UX:**
    * Sticky Navbar with **Glassmorphism** effect on scroll.
    * Smooth hover effects and transitions.
    * Interactive "Add to Cart" feedback.
* **🏗️ Solid Structure:**
    * **Semantic HTML5** tags for better SEO and Accessibility.
    * **CSS Grid & Flexbox** layouts.
* **⚡ Organized Codebase:**
    * Variables for colors and typography management.
    * Modular CSS structure.
